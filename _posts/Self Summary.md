# Self Summary



## 自我感受

```
  现在的感受非常的简单，就是被辞退了，很失落，颓废感。人生要经历很多的大起大落，这也是对我的一个考验，有这个结果也是自己当初愿意来这家公司的原因，归根究底都是自己的选择，怨不得别人。我从米开朗基杨的文章得到了感悟，总结一下几点：
1.认清自己不足
2.坚持去做，没有实际行动的自我总结是失败的
3.勇敢去做
  虽然被解除劳动合同，让人很悲伤，但是我在公司里认识way，jiran，carl，mitt，perror这些朋友，其中way 给我的帮助是巨大的，我现在的技术走向以及能力的提升和他对我的指导是分不开的，即使不是同事了，我们依然是非常棒的朋友。
  悲伤过后就是自我鼓励和监督，要开始提升自己，下面就要开始从技术，人际交往两个方面进行总结，并明确今后该怎么去做。
```
